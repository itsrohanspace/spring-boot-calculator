# Calculator Application using Spring Boot

This is a simple Calculator Application built using Spring Boot.
The project includes REST APIs, a web-based calculator using Thymeleaf,
and a full stack calculator using JavaScript.

## Features
- Addition, Subtraction, Multiplication, Division
- REST API based calculator
- Web calculator using Thymeleaf
- Full Stack calculator using JavaScript

## Technologies Used
- Java 17
- Spring Boot
- Spring Web
- Thymeleaf
- HTML, CSS, JavaScript
- Maven

## API Endpoints
- /calc/add?a=10&b=5
- /calc/sub?a=10&b=5
- /calc/mul?a=10&b=5
- /calc/div?a=10&b=5

## How to Run the Project
1. Clone the repository
2. Open the project in STS or IntelliJ IDEA
3. Run `CalculatorApplication.java`
4. Open browser and visit:
   http://localhost:8080
